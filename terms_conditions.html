About MindStark Library

MindStark Library is an online reading platform that provides access to public domain books, curated reading tools, and premium membership features. Our core book catalogue is sourced from Project Gutenberg — a free, public domain digital library — via the Gutendex API.

What we provide:

  • Access to 70,000+ free public domain books via Project Gutenberg
